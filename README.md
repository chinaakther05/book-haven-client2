# üìö Book Haven

Book Haven is a full-stack web application where users can explore, add, update, and delete their favorite books. Users can also leave reviews for books, and the home page displays the top-rated books.  

---

## üìù Features

- **Authentication**: Login and registration with Firebase Authentication.  
- **My Books**: Users can add, view, update, and delete their own books.  
- **Reviews**: Users can leave short text reviews for each book. Reviews are stored in MongoDB and displayed dynamically.  
- **Top Rated Books**: Displays the top 3 books with the highest ratings using MongoDB sorting.  
- **Dark Mode Support**: Supports both light and dark themes.  
- **Responsive Design**: Works smoothly on both desktop and mobile devices.
